# Variable Definitions

This file defines the primary variable used in the behavioral analysis of the YRBS 2007 dataset.

- **SadOrHopeless**: This variable measures adolescent mental health. It represents whether the student felt so sad or hopeless almost every day for two weeks or more in a row that they stopped doing some usual activities during the past 12 months.
    - **Code 1**: Yes (Indicates presence of persistent sadness/hopelessness)
    - **Code 2**: No (Indicates absence of persistent sadness/hopelessness)

*Note: This analysis focuses specifically on the psychological well-being variable as the primary indicator for our study.*
