# Variable Definitions

## 1. Behavior Variable: SadOrHopeless
* **Description**: During the past 12 months, did you ever feel so sad or hopeless almost every day for two weeks or more in a row that you stopped doing some usual activities?
* **Type**: Categorical (Binary)
* **Original Coding**: 1 = Yes, 2 = No
* **Benchmark**: $p_0 = 0.30$

## 2. Continuous Variable: BMIPCT
* **Description**: Body Mass Index (BMI) percentile based on sex and age.
* **Type**: Continuous
* **Range**: 0 to 100
* **Benchmark**: $\mu_0 = 65.0$

## 3. Demographic Variable: WhatIsYourSex
* **Description**: The biological sex of the respondent.
* **Type**: Categorical (Binary)
* **Original Coding**: 1 = Female, 2 = Male
